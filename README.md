# CalculatorProject

## Description
This is a Java Console Calculator project built as part of the Java Developer Internship task.  
It allows users to perform addition, subtraction, multiplication, and division.  

## Features
- Addition, Subtraction, Multiplication, Division
- Handles divide-by-zero errors
- Loops until user chooses to exit
- Uses methods for each operation
- Takes input via `Scanner`
